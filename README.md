# Homebrew Tap

Homebrew formulae for [cons](https://github.com/mattsonlyattack/construct-app) - a structure-last personal knowledge management CLI.

## Installation

```bash
brew install mattsonlyattack/tap/cons
```

Or add the tap first:

```bash
brew tap mattsonlyattack/tap
brew install cons
```

## Upgrade

```bash
brew update
brew upgrade cons
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| cons | Structure-last PKM CLI with AI-powered auto-tagging |
